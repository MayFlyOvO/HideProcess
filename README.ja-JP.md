# BossKey

[English](README.md) | [简体中文](README.zh-CN.md) | 日本語

<sub>この種のツールとしては珍しく、UI と操作感まで丁寧に整えた Windows Boss Key ツールです。</sub>

BossKey は、指定したアプリケーションウィンドウをすばやく非表示・復元・整理するための Windows デスクトップツールです。いわゆる "boss key" 用途だけでなく、日常作業向けの軽量なウィンドウ管理ツールとしても使えます。

![BossKey メインウィンドウ概要](docs/images/main-window-overview.png)

## 基本ワークフロー

- 選択したターゲットウィンドウをまとめて即座に非表示 / 復元
- 実行中ウィンドウ一覧からターゲットを追加可能
- ウィンドウピッカーで直接ウィンドウを選択して追加可能
- ターゲットのグループ化、グループ名変更、折りたたみ、ドラッグによる移動に対応
- グローバルホットキーとグループ別ホットキーに対応
- キーボードのみのホットキーに加え、`Ctrl + Alt + Mouse Middle` のようなキーボード + マウスボタンの組み合わせにも対応
- 非表示 / 復元に同じホットキーを使う `Toggle` モードに対応

## ターゲットごとの動作設定

各ターゲットは右クリックメニューから個別に設定できます。すべてのアプリに同じルールを適用する必要はありません。

- ターゲットの有効 / 無効
- 非表示時にミュート
- 非表示時にフリーズ
- 表示時に最前面化
- 表示時にマウス位置の中央へ移動

![BossKey ターゲット右クリックメニュー](docs/images/target-context-menu.png)

## 設定ウィンドウ

設定ウィンドウでは、ホットキー、グループ挙動、言語、更新チェック、テーマ入口をまとめて管理できます。

- グローバルホットキー
- グループ別ホットキー
- 起動時動作とトレイ動作
- 更新チェック
- 言語選択
- テーマ設定入口

![BossKey 設定ウィンドウ](docs/images/settings-window.png)

## テーマシステム

BossKey には、システム既定に頼らない内蔵テーマシステムがあります。

- 既定のライト / ダークテーマ
- 編集中のリアルタイムプレビュー
- テーマカラーのカスタマイズ
- テーマ対応メッセージボックス / ダイアログ

![BossKey テーマ設定ウィンドウ](docs/images/theme-settings-window.png)

![BossKey カラーピッカー](docs/images/color-picker-window.png)

## 言語パック

- 内蔵言語: English
- 現在このリポジトリに含まれる追加言語パック: 簡体字中国語、日本語
- インストール済み言語パックの保存先:
  - `%APPDATA%\BossKey\Languages`
- メイン設定ファイルの保存先:
  - `%APPDATA%\BossKey\settings.json`
- アプリ更新チェック時には、インストール済み言語パックの新しいバージョンも確認できます

## 動作環境

- Windows 10 / 11
- x64
- ソースからビルドする場合は .NET 8 SDK が必要

## ローカル実行

```powershell
dotnet build BossKey.sln
dotnet run --project .\BossKey.App\BossKey.App.csproj
```

## リリースビルド

```bat
Build-Release.bat
```

GitHub Actions では以下をビルドします。

- 自己完結型インストーラー
- 自己完結型単一ファイル版

## 注意事項

- 権限の高いプロセスや保護されたプロセスでは、フリーズ機能を安定して使うために BossKey 自体を管理者として実行する必要がある場合があります
- マルチプロセスアプリの動作は、選択したウィンドウを実際に所有しているプロセスに依存します
- 一部のシステムウィンドウ、UWP コンテナーウィンドウ、特殊描画ウィンドウは、通常のデスクトップアプリと同じように復元できない場合があります
